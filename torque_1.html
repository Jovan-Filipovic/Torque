1. Input Data

Variable Value Unit Description
D: mm Bolt Diameter (metric)
P: mm Bolt thread
z: - Nummber of thread per coil
s: mm Wrench size
D0: mm Bolt hole diameter
α deg Thread angle
μ - Friction factor for thread (copper lubricant)
μnut - Friction factor for nut(dry surface, mashined)
Re: MPa Yield stress of bolt
φ - Near yield limit ratio

d3: mm Min. root thread diameter
φ: rad Thread coil angle
ρn: rad Thread friction angle
rμ: mm Wrench friction diameter
As: mm2 Bolt Stress calculation area
F: N Bolt tension Axial Force
Tn: Nm Torque for friction on thread
Tμ: Nm Torque for friction under the nut
T: Nm Total Torque required
